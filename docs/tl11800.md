## Formulario principal

O HTML logado foi capturado por "Salvar pagina completa" no Chrome. O formulario
principal e:

```text
form: frmtl11800
action: http://3.19.17.18//telas/tl11800.asp
method: post
```

Campos tecnicos principais:

- O.S: `txtnr_ordserv`
- Veiculo: `txtcd_veiculo`
- Placa: `txtplaca`
- Componente: `txtcd_compo`
- Entrada Data/Hora: `txtdh_entrada`
- Entrada Horimetro: `txtqt_hr_ent`
- Entrada Km: `txtqt_km_ent`
- Saida Data/Hora: `txtdh_saida`
- Saida Horimetro: `txtqt_hr_sai`
- Saida Km: `txtqt_km_sai`
- Data Prevista para Inicio: `txtdh_inicio`
- Data Prevista de Liberacao: `txtdh_prev`
- Horas Previstas: `txtqt_hr_prev`
- Horas Realizadas: `txtqt_hr_rea`
- Filial: `txtcd_filial`
- Departamento: `txtcd_ccusto`
- Motorista: `txtcd_moto`
- Contrato No: `txtcd_contrat`
- VL Acrescimo: `txtvl_acresc`
- O.S Retorno: `txtnr_os_ret`
- Observacao: `txtnm_observ`

Flags:

- Investimento: `chkbl_invest`
- Acidente: `chkbl_acident`
- Socorro: `chkbl_socorro`
- Retorno: `chkbl_retorno`
- Programada: `chkbl_corre`

Fluxo observado:

- `Novo` seta `hidstatusreg = 4` e envia o formulario.
- `Salvar` valida campos e envia o mesmo formulario.
- `Excluir` seta `hidstatusreg = 3`.

As telas de reclamacao/reclamacao livre sao modais separadas:

- Reclamacao: `tl11804.asp?cd_empresa=1&nr_ordserv=<OS>`
- Reclamacao Livre: `tl11802.asp?cd_empresa=1&nr_ordserv=<OS>`
